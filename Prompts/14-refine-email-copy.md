# Prompt: Refine / Rewrite Email Copy

> Use this prompt to improve the text content of an HTML email — making it more engaging, concise, or action-oriented.

---

## Prompt

```
Improve the text copy in this HTML email. Here's what I need:

**Current email purpose:** [e.g., "Promote a college cultural fest and drive registrations"]

**Target audience:** [e.g., "College students aged 18-22, IIIT Nagpur campus"]

**Tone:** [e.g., "Exciting, energetic, FOMO-inducing, slightly informal but not sloppy"]

**What to improve:**
1. **Headlines** — Make them punchy and attention-grabbing. Use power words. Keep them short.
2. **Body text** — Make it concise. Cut fluff. Every sentence should either inform or persuade. Aim for scannable text (short paragraphs, 2-3 lines max).
3. **CTA text** — Make button text action-oriented and specific (e.g., "Register Now" > "Click Here", "Get Your Pass" > "Submit").
4. **Urgency/scarcity** — Add subtle urgency where appropriate (dates, limited spots, deadlines) without being spammy.
5. **Flow** — Ensure the email follows a logical flow: hook → value → proof → CTA.

**Constraints:**
- Do NOT change the HTML structure, only the text content.
- Keep the same sections and order.
- Maintain existing formatting (bold, line breaks, etc.)
- Stay within similar character counts (don't make sections drastically longer or shorter).
- Return the updated text for each section clearly labeled.

**Current email text:**
[Paste the text content of each section, or paste the full HTML]
```
