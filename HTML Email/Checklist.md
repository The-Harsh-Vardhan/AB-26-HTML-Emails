Alright—before you hit Send, run through this quick 2-minute pre-send checklist. It’s the kind of simple discipline that separates a smooth launch from a “whoops, wrong link” moment. Think of it as checking the ropes and lights before the circus curtain rises. 🎪✨

📧 1. Links & Buttons

Primary CTA (“🎟 Get Your Pass”) → opens the passes page correctly.

Explore Events → goes to the events listing.

Website link → loads on both desktop and mobile.

Any brochure/download links → actually download the file.

Tip: click each link from a phone too, not just desktop.

🖼 2. Images Load Properly

Banner and hero image show without requiring login.

Image links are public URLs (not local files or private Drive links).

File size ideally < 300 KB per image so emails load quickly.

📱 3. Mobile View Check

Open the email on a phone and confirm:

Text is readable without zoom.

Buttons are big enough to tap.

Images scale correctly (no sideways scrolling).

🖥 4. Desktop View Check

Look at it in a normal desktop client:

Layout centered around ~600–620px width.

Buttons appear side-by-side as intended.

Fonts and spacing look consistent.

📨 5. Subject & Preview

Confirm:

Subject line: clear and exciting.

Preview text: complements the subject.

Example:

Subject: 🎭 The Curtain Rises: Abhivyakti ’26 Registrations Now Open for IIITN

Preview: One pass. Unlimited events. Pro shows, pro nights, and exclusive merch await.

🔗 6. Login Instruction Is Clear

Students must know to use their IIITN email. Make sure the note appears:

The IIITN Pass will only be visible when you log in using your IIIT Nagpur student email ID.

🧪 7. Send a Test Email

Send it to:

yourself

one Android phone

one iPhone if possible

Check:

loading speed

button alignment

any strange formatting.

⚖️ 8. Keep Email Size Safe

If the message is too large, Gmail may clip it.

Limit images to 2–4 main visuals.

Avoid huge GIFs.

🕒 9. Timing

If you’re sending to students, good open-rate windows are usually:

Morning: 9–11 AM

Evening: 6–9 PM

📢 10. Reinforce the Email

Right after sending:

Post the same CTA link in WhatsApp groups.

Share the poster + link on Instagram stories.

That way the email isn’t the only channel—it becomes part of a coordinated announcement wave.
